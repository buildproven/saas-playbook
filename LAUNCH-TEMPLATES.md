# Launch Templates

Ready-to-adapt social and email templates for shipping a developer tool or SaaS product. Find and replace all `[BRACKETED]` placeholders before posting.

## Placeholders

| Placeholder | What it is |
|-------------|------------|
| `[PRODUCT_NAME]` | Your product name |
| `[PRODUCT_URL]` | Your product page URL |
| `[KEY_FEATURE_1..4]` | Main selling points |
| `[TIME_SAVED_FEATURE_1]` | "~4 hours" etc. |
| `[STACK]` | Tech stack list |
| `[PRICE_SINGLE]` | Single license price |
| `[PRICE_UNLIMITED]` | Unlimited license price |
| `[YOUR_NAME]` | Your first name |
| `[BRAND_NAME]` | Your brand/company name |
| `[PAIN_POINT]` | The thing you were tired of repeating |
| `[LAUNCH_PRICE_ENDS]` | "Jan 1" etc. |
